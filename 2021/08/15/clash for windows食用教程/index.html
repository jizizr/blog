clash for windows食用教程

发表于 2021-08-15 更新于 2021-08-17

这篇文章我会从下载、安装、添加订阅和导入的方式来详细讲述如何使用clash for windows

下载

下载主要有两个方式，一个是从官方的Github页面下载、一个是我的节点站

官方Github页面

  1. 打开https://github.com/Fndroid/clash_for_windows_pkg/releases
  2. 选择Clash.for.Windows.Setup.0.17.1.exe或Clash.for.Windows.Setup.0.17.1.ia32.exe文件下载

节点站

  1. 打开节点站
  2. 点击clash for windows x64或clash for windows x86进行下载

安装

  1. 打开安装包
  2. 一路next安装

使用

  1. 使用下方任一方式导入节点
  2. 右键小猫咪图标
  3. 勾选“System Proxy”

  1. 进入主页，选择Proxies
  2. 选择 Rule （规则） 或 Global （全局）
  3. 选择节点或使用默认的自动选择节点

  1. 记得科学上网完后，退出clash for windows

使用订阅获取节点

  1. 打开 clash for windows
  2. 转到 Profiles
  3. 粘贴订阅链接，点击Download

  1. 点击设置图标

  1. 设置别名和更新时间间隔

配置文件

一般Clash的配置文件为yml或yaml为后缀

  1. 打开 clash for windows
  2. 转到 Profiles
  3. 点击 Import

  1. 选择文件

  1. 切换配置文件
